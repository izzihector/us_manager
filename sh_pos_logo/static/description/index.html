Point Of Sale Logo

This module helps to set the global POS receipt logo and global header logo which is applied for all the POS sessions. You can set the custom POS receipt logo and custom header logo for the specific POS session also.

How it works? click here to watch video

Features

Easy to set the custom logo in the point of the sale receipt and point of sale header.

You can set the global logo in the point of the sale receipt and point of sale header.

No special configuration required just install it and cheers!

This module saves your important time.

It reduces human efforts.

Tick the checkbox "Enable POS Receipt Logo" to set the custom logo for the receipt and tick the checkbox "Enable POS Header Logo" to set the custom logo for the header.

Print Product Labels via Zebra Printer with Configuration.

Odoo Extension Allowing The User To Print Product Label Directly by configure the dimesnsion and margins of Barcode Via Zebra Printer Instead Of Downloading As a PDF File.

This module uses the QZ Tray client for sending direct print commands to the printer:

  • Installation Link: (https://qz.io/download/)
  • Demo Link: (https://demo.qz.io/)

Salient Features

  • User can select the default Zebra printer for a report.
  • It will print the Product Name along with the Barcode of Product.
  • User can Configure the Label Height, Width, Left Margins, Buttom Margins.
  • User can also Configure the Product Name Height, Width, Left Margins, Buttom Margins.
  • User Can Change the position of product name for ZPL printing.

Setup& Configuration

Configuring Printers

This module depends upon another module "Print Odoo Reports via Zebra Printer by Webkul Software Pvt. Ltd."

You can download the module from link: https://apps.odoo.com/apps/modules/12.0/wk_odoo_directly_print_reports/

After the successful installation of both the modules You can setup the available printers under the Settings > Technical menu.

Enter the details of printer.

  • Printer Name: Name of your printer.

    Make sure the Printer's name is exactly the same as the name of the printer is used by the QZ Tray client to make connection and send print commands.

  • Printer type: Select the printer type here. Currently, only zebra printers are supported.

Odoo-Report Configuration.

Next, you'll need to configure the Product Barcode (ZPL) report action in the report that you wish to send to the printer. You can set this from the Settings > Technical menu under the Reporting section.

In the Product Barcode (ZPL) report form view, you can set the following parameters:

  • Report Action: Select the "Send To Printer" option here if you wish to print the report directly instead of downloading it. The default value of this field is "Default Odoo's Action" that leads to the default Odoo's behaviour.
  • Printer: Select the default printer here. Leave this field blank if you wish to select the printer from the list of available printers while printing it.
  • Use Template:Always Select this option.
  • Report Template: Select the Product Barcode Report Template record here.

Configuration for Label.

  • You will find the tab Product Barcode Settings in which you will get multiple fields for Configurations.

    • Barcode Height.
    • Barcode Width.
    • Barcode Bottom Margin.
    • Barcode Left Margin.
      • You can change the values of all the fields as per your need.

  • You will also got a tab Product Name Settings in which you will get multiple fields for Configurations.

    • Product Name Height.
    • Product Name Width.
    • Product Name Bottom Margin.
    • Product Name Left Margin.
      • You can change the values of all the fields as per your need.

  • You will also find a tab Report Template which have the zpl of Product Label.

Work Flow.

  • After successful configuration of printer and Report template print Product label by clicking on Product Label (ZPL Direct) under the Print button on Product Variants.

Odoo 13 POS Custom Discount

Apply special discount for product or on the order from point of sale interface.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • First we have to configure the rule for discount to use prefixed discount.


  • Enable "Discount" as following.


  • To open discount rules click on disc button or press the character which is configured in POS configuration.
  • In this window user can select discount by up and down key or by click on it and for apply that discount by press 'S' or 'A' or by click on button.


  • After Apply discount it added to item.


  • User can also give a custom discount to customer which is not configure.
  • To open custom discount popup user need to click on manual or press 'M' from keyboard.




  • After apply manual discount it will added to item with discount note.
